# Simple-educational-keylogger
ello everyone, this is just a simple Keylogger creating a Keystrokes file everytime you start it.
It only goes for 1 Minute so ist very simple. 



Folder should contain: Keylogger.py
                       keystrokes.log
                       readme


1. Install python on youre pc, go on this Website: https://www.python.org/downloads/
2. Go into youre cmd and write following code: cd (here the path of the downloaded Folder)
   Example: cd Desktop/Keylogger   (if extracted Folder is on youre Desktop)
3. go into the cmd and write following code: python Keylogger.py
   The Keylogger will only run for 60 seconds, you can easily extand or decreas the duration in the py file.
4. now you can type everywhere on youre Personal Computer, and look it up in the new created file called "keystrokes"
